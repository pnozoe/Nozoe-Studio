01

Color

Una paleta dual: tinta navy como ancla estructural, naranja cálido como único acento. Los neutros se construyen sobre matices fríos (sky) y cálidos (warm) para evitar grises planos. El acento nunca supera el 5% de cualquier superficie.

05

Movimiento

Curvas precisas, nunca rebotantes. La duración base es 250ms; usar más rápido para hover/feedback inmediato y más lento solo para transiciones de página.
